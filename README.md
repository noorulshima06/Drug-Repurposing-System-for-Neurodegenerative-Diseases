# Drug-Repurposing-System
## Overview
This project id an AI Driven Drug Repurposing System developed for neurodegenerative diseases such as Alzheimer's and Parkinson's diseases.
The system analyzes gene similarity and predicts potential drug candidates using Machine Learning techniques.
## Technology Used
Python,
Machine Learning,
Logistic Regression,
Jaccard Similarity,
Pands,
NumPy,
Scikit-learn.
## Features
Drug candidate prediction,
Gene similarity analysis,
Disease dataset processing,
AI-based prediction system,
Neurodegenerative diseases analysis.
## Project Files
DrugRepurpose.ipynb -> Main project notebook,
README.md -> Project documentation.
